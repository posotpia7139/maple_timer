# 메이플 심플 타이머

메이플스토리 사냥 회수 체크를 빠르게 사용할 수 있도록 만든 브라우저 기반 타이머입니다.  
메인 타이머와 보조 타이머를 함께 운용할 수 있고, 마지막 설정값은 브라우저에 저장됩니다.

## 주요 기능

### 1. 메인 회수 타이머
- 기본값은 `100초(1분 40초)`로 설정됩니다.
- 남은 시간을 큰 카운트다운으로 표시합니다.
- 경과 시간은 별도로 누적 표시합니다.
- 원형 프로그레스 링으로 현재 사이클 진행 상태를 시각적으로 확인할 수 있습니다.

### 2. 시작 / 초기화
- `START` 버튼으로 타이머를 시작합니다.
- 동작 중에는 버튼이 `RESET`으로 바뀌며, 다시 누르면 전체 타이머가 초기화됩니다.
- 키보드 `Space`, `Enter`, `Numpad Enter`로도 시작/초기화할 수 있습니다.
- 입력창에 커서가 있을 때는 단축키가 동작하지 않도록 처리되어 있습니다.

### 3. 보조 타이머
- 우측 패널에서 보조 타이머를 여러 개 추가할 수 있습니다.
- 각 보조 타이머는 이름, 시간, 활성화 여부를 개별 설정할 수 있습니다.
- 이름은 공백을 포함해 입력할 수 있으며 최대 10자까지 지원합니다.
- 시간은 초 단위 숫자 또는 `MM:SS` 형식으로 입력할 수 있습니다.
- 각 보조 타이머는 별도의 진행 바를 표시합니다.
- 삭제 버튼으로 개별 보조 타이머를 제거할 수 있습니다.

### 4. 보조 타이머 패널 제어
- `보조 타이머` 버튼으로 우측 패널을 열고 닫을 수 있습니다.
- 패널을 닫으면 보조 타이머는 비활성화되고, 다시 열면 이전 활성 상태를 복원합니다.
- 리스트 영역은 스크롤 가능하지만 스크롤바 UI는 보이지 않도록 구성되어 있습니다.

### 5. 사운드 알림
- 메인 타이머가 한 사이클을 완료하면 알림음이 재생됩니다.
- 실행 중에는 별도의 랜덤 비프음이 주기적으로 재생됩니다.
- 보조 타이머도 각자 주기 완료 시 알림음을 재생합니다.
- 상단 볼륨 슬라이더로 전체 알림음 크기를 조절할 수 있습니다.

### 6. 설정 자동 저장
- 아래 항목은 브라우저 `localStorage`에 자동 저장됩니다.
- 메인 타이머 시간
- 볼륨 값
- 보조 타이머 패널 열림/닫힘 상태
- 보조 타이머 목록, 이름, 시간, 활성화 상태

## 사용 방법

1. 메인 타이머 시간(초)을 입력합니다.
2. 필요하면 보조 타이머 패널을 열고 항목을 추가합니다.
3. 각 보조 타이머의 이름과 시간을 설정합니다.
4. `START` 버튼을 눌러 전체 타이머를 시작합니다.
5. 종료하거나 다시 맞추고 싶으면 `RESET`을 누릅니다.

## 파일 구성

- `timer.html`: 타이머 UI 구조
- `styles.css`: 전체 스타일 및 인터랙션 스타일
- `script.js`: 메인 타이머, 보조 타이머, 사운드, 설정 저장 로직

## 실행 방법

별도 빌드 없이 정적 파일로 동작합니다.  
`timer.html`을 브라우저에서 열면 바로 사용할 수 있습니다.
